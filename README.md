# 🚗 Obstacle Avoidance Robot Car

## 📌 Overview
This project is an autonomous robot that detects obstacles and avoids them using an ultrasonic sensor.

## ⚙️ Components Used
- ARDUINO UNO
- Ultrasonic Sensor (HC-SR04)
- Motor Driver (L298N)
- DC Motors
- Battery Pack

## 🔌 Circuit Diagram
https://drive.google.com/file/d/1VP-BGRt_M2pjHPiFPh8hS8NqGU8guRpm/view?usp=drivesdk

## 💻 Installation
1. Clone the repository
2. Open code in Arduino IDE
3. Upload to ARDUINO UNO

## 🚀 Working Principle
- Measures distance using ultrasonic sensor
- If obstacle detected → stops
- Changes direction automatically
